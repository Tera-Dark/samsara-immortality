# 轮回·仙途

一款修仙人生模拟游戏。核心体验围绕开局命格、成长抉择、地图探索、人物关系、境界突破与轮回展开。

## 当前状态

当前版本：`0.9.0`

已经具备公开试玩所需的核心闭环：

- 开局随机资质、命格与成长路线
- 主线任务与阶段引导
- 地图、地点、NPC 与地点交互
- 角色信息、背包、功法、任务、图鉴
- 战斗、修行、突破、闭关、关系经营
- 多槽位存档、导入导出

## 启动方式

```bash
npm install
npm run dev
```

构建发布包：

```bash
npm run build
```

## 操作说明

- `Space`：重复上一次行动
- `Ctrl + E`：打开事件编辑器
- 点击顶部任务卡：查看任务详情
- 点击角色或 NPC 卡片：打开详细信息
