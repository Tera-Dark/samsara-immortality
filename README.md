## 🌌 关于游戏

《轮回·仙途》是一款高自由度的**文字沙盒修仙人生模拟器**。
我们抛弃了传统修仙游戏枯燥的“挂机点数值”和“线性走剧情”模式。在这里，你将体验真正的**“百态人生，动态演算”**。

在看似静谧的文字排版下，隐藏着一个复杂的、基于事件树与实体组件（ECS）架构驱动的修仙界。你的出生、资质、每一次奇遇的选择、甚至仅仅是岁月的流逝，都在默默推动着命运齿轮的转动。

寿元将尽时，是选择冒险强行破境，还是安详晚年，留待下一世轮回再续仙缘？

---

## 🌟 核心特色

### 🦋 蝴蝶效应般的事件网络
没有固定的主线任务和剧本，只有基于你当前状态实时演算的**条件触发式事件**。
- **生而不同**：每一次轮回，你的“五维属性”、“初始天赋卡包”、“命途”完全随机。
- **因果相连**：幼年时在深山的一场玩雪，可能导致体质虚弱；炼气期在坊市的偶然结怨，可能会在几十年后引来仇家截杀。每一次选择都会永久改变你的“因果标记”。

### 🌍 动态演化的长生界
引擎每次开局都会自动生成广袤的修仙世界，包括城镇、坊市、秘境以及门派。
- **活着的 NPC**：世界中游荡的不仅是“杂鱼怪”，而是拥有独立姓名、寿命、境界与背景的修仙者。他们同样在修炼、老去。你的实力决定了你能与他们产生何种交集（交谈、切磋、双修、截杀）。

### ⚖️ 硬核修仙养成系统
- **五维抉择**：体魄、悟性、资质、魅力、气运。
- **功法与境界**：打坐吐纳、闭关破境寻找突破契机。
- **时间管理**：修仙无岁月，一晃数十载。有限的寿元是你最大的敌人，合理安排每一次游历与闭关的时间，在天人五衰降临前夺天地造化。

### 🛠️ “天道”降临：内置创世编辑器
作为一款纯正的 Meta 游戏，我们赋予玩家修改“天道法则”的权利。
- 按下 **`Ctrl+E`** 即可在游戏内一键唤出**全功能可视化事件编辑器**。
- 无需懂得编程，动动鼠标即可创建新的奇遇、调整天命几率、添加复杂的网状分支剧情。你的每一次创造，都能即刻应用于当前的轮回世界中。

## 🎭 开发愿景

我们是一个独立游戏团队，致力于将“修仙模拟”推向更深的维度。《轮回·仙途》目前处于 **Alpha 测试阶段**。核心引擎骨架已经搭建完毕（极其强健的 React+TS 数据驱动方案），后续我们将持续填补：
- 庞大的奇遇文本库。
- 更多维度的人际交往（结缘、收徒、宗门大战）。
- 阵法、炼丹、炼器等辅修百艺系统。

修仙本是逆天而行，愿各位道友在此找到属于自己的长生大道。
